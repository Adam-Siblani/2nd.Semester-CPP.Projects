# 2nd.Semester-CPP.Projects
This repository showcases my progress in C++ programming during the second semester of my IT studies.
I have recently completed my second semester at one of Europe's leading engineering universities.
It contains 5 academic projects assigned throughout the semester, presented in chronological order to clearly demonstrate my growth in object-oriented programming, software design, algorithmic thinking, and modern C++ development.
The entire repository was built using a pure Linux environment, with all project directories and documentation created directly from the terminal. This reflects my practical experience with Linux systems, command-line workflows, Git version control, and structured software development.
